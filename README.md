# Static Next Interator Frontend

Skill e microfilosofia para um estilo de frontend estatico em JavaScript com:

- `bootApp` central
- componentes com contrato `next(message = {})`
- render em HTML string
- eventos delegados por `data-cid` e `data-message`
- props e tools injetados a partir do runtime

## Conteudo

- `SKILL.md`: skill pronta para uso no Codex
- `philosophy.md`: principios de arquitetura
- `examples/`: exemplos curtos de componentes, eventos e bootapp
- `references/`: contrato e guia de eventos

## Uso

Copie a pasta para `.codex/skills/static-next-interator-frontend/` ou leia o `SKILL.md` como base para projetos no mesmo estilo.
