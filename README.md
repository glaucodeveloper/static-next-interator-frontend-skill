# Static Next Interator Frontend

Skill e microfilosofia para um frontend estatico em JavaScript com tres camadas claras:

- componentes para estado local e renderizacao de UI
- interators para eventos, estado compartilhado e coordenacao
- generators para producao passo a passo do programa

## Conteudo

- `SKILL.md`: skill pronta para uso no Codex
- `philosophy.md`: principios de arquitetura
- `examples/`: exemplos curtos de componentes, eventos, bootapp e generator
- `references/`: contrato, eventos, interators e producao de programa

## Uso

Copie a pasta para `.codex/skills/static-next-interator-frontend/` ou leia o `SKILL.md` como base para projetos no mesmo estilo.
