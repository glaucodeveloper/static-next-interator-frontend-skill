# Anúncio para a comunidade

**Static Next Interator Frontend** propõe componentes React-like em JavaScript puro com uma anatomia explícita:

```text
handler -> this.update(id, patch) -> state -> this.template(id) -> outerHTML
```

Cada módulo registra instâncias em `this.frontends`, callbacks em `this.events`, HTML em `this.template()` e transições em `this.update()`. O projeto inclui um contador canônico, exemplos progressivos de UI e uma página interativa que explica cada parte.

Projeto: https://github.com/glaucodeveloper/static-next-interator-frontend-skill
